# Clínica Piedrazul - Angular Application

Este proyecto es una modernización de la landing page original de la **Clínica Piedrazul**, migrada de una estructura estática a una arquitectura robusta basada en **Angular**.

## 🌟 Características Principales

- **Arquitectura Basada en Componentes**: Separación clara de responsabilidades (Header, Navbar, Hero, Doctors, Registration, Footer).
- **Diseño Premium (UX/UI)**: Implementación de tendencias modernas como **Glassmorphism**, sombras suaves (Soft-UI) y micro-interacciones CSS.
- **Formularios Reactivos**: Sistema de registro validado en tiempo real con `ReactiveFormsModule` de Angular y retroalimentación visual mediante **Bootstrap**.
- **Responsive Design**: Adaptabilidad total a dispositivos móviles, tablets y escritorio.
- **Notificaciones Dinámicas**: Integración de **Toastify.js** para confirmaciones de usuario con estilo profesional.

## 🛠️ Tecnologías Utilizadas

- **Angular 19+** (Standalone Components)
- **TypeScript** (Lógica de negocio y validaciones)
- **Bootstrap 5** (Layout y grillas)
- **Font Awesome 6** (Iconografía médica y social)
- **Toastify.js** (Sistema de notificaciones)
- **CSS3 / HTML5** (Estilos personalizados y estructura semántica)

## 🚀 Instalación y Ejecución Local

Para correr este proyecto en tu entorno local, asegúrate de tener instalado [Node.js](https://nodejs.org/) y el Angular CLI.

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/Julianste15/piedrazul-angular.git
   cd piedrazul-angular
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   ng serve
   ```
   Abre tu navegador en `http://localhost:4200/`.
